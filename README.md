# 📝 Blog App (Next.js)

A simple blog website built with Next.js where users can browse and read posts.

🔗 Live Demo: https://blog-website-next-js-delta.vercel.app/

---

## 🚀 Features

- Blog listing page
- Blog detail page (dynamic routes)
- Search posts
- Loading states
- 🌗 Dark / Light theme support (next-themes)
- Error handling
- Basic SEO setup

---

## 🧠 What I Learned

- Next.js routing (including dynamic routes)
- Fetching data from API
- Server components
- Handling loading and errors
- Deploying with Vercel

---

## 🔧 Tech Stack

- Next.js
- React
- Tailwind CSS
- JSONPlaceholder API

---
